## Description

## Usage

**Do** use the `Button` for single occasions as well as multiple ones in a row.
It is suitable for toolbars.

**Do** use the `Button` for asynchronous calls. You can reflect a loading state
by setting it to `disabled` and rendering a loading animation, then switch back
to the original content when done.

**Do not** mix small and normal sized `Button`s. Keep horizontal and vertical
alignments.

**Do not** combine the `Button` in junction with other controls such as inputs.
It is okay to put other controls nearby. Leave space in that occasion.
